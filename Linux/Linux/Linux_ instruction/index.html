命令通用格式: command [-options] [parameter]
—例如

ls命令

ls命令的作用是列出目录下的内容,默认以平铺形式呈现当前目录内容

用法:ls [-a -l -h] [linux路径]

  • -a 将所有文件展示出来，包含隐藏文件
  • -l 将文件以列表形式展现
  • -h 表示以易读形式K/M/G列出文件大小，必须搭配-l一起使用

命令选项是可以组合使用的，如ls -l -a /home

cd、pwd命令

mkdir命令

创建新的目录/文件夹

语法：mkdir [-p] Linux路径

  • 参数必填表示要创建文件夹的Linux路径
  • -p可选，表示自动创建不存在的父目录

支持相对路径进行目录创建

touch、cat、more命令

touch

通过touch命令创建文件

语法：touch Linux路径

  • touch无选项，参数必填

cat

查看文件内容

语法：cat Linux路径

  • 没有选项，参数必填

more

同样是查看文件内容，与cat命令不同的是cat命令直接将内容全部展现出来，more命令支持翻页

语法：more Linux路径

  • 没有选项，参数必填
  • 查看时通过空格翻页
  • 通过q退出查看

cp、mv、rm命令

cp

用于复制文件/文件夹

语法：cp [-r] 参数1 参数2

  • -r：用于文件夹的复制,包括里面的文件
  • 参数1表示被复制的文件/文件夹(Linux路径)
  • 参数2表示复制去的地方(Linux路径)

mv

用于移动文件/文件夹

语法：mv 参数1 参数2

  • 参数1：Linux路径，表示被移动的文件/文件夹
  • 参数2：Linux路径，表示要移动的地方，不存在则会将文件进行改名

rm

用于删除文件/文件夹

语法：rm [-r -f] 参数1 参数2 .... 参数N

  • -r：同cp里的-r一样对象为文件夹，用于文件夹删除
  • -f:表示强制删除
  • rm命令支持通配符

which、find命令

which

查找命令本体所在的文件

语法：which 要查找的命令

find

查找指定的文件

-name

语法：find 起始路径 -name 被查找的文件名

  • 查找文件时可以使用通配符

-size

查找文件大小大于/小于n[kMG]的文件

语法：find 起始路径 -size +|- -n[kMG]

  • 注意k为小写

grep、wc命令

grep

在文件中通过关键字过滤文件行

语法：grep [-n] 关键字 文件路径

  • -n表示在结果中显示匹配行的行号

wc

统计文件的行数、单词数等

语法：wc [-c -m -l -w] 文件路径

  • -c：统计bytes数量
  • -m：统计字符数量
  • -l：统计行数
  • -w：统计单词数量

管道符

将左边的结果(能产生内容输出)作为右边的输入

echo、tail、重定向符命令

echo

在命令行中输出指定内容

语法：echo 输出的内容

  • 被``包括的内容会被当作命令执行输出

重定向符

  • >表示将左侧命令结果覆盖写入右侧指定文件
  • >>表示将左侧命令的结果追加写入右侧指定的文件

tail

查看文件尾部内容，跟踪文件最新修改

语法：tail [-f -num] Linux路径

  • -f表示持续追踪
  • -num表示查看尾部多少行，不填默认10行

vi/vim编辑器

文本编辑器，可以在命令行中进行文本编辑

语法：vim 文件路径

  • 如果文件不存在，会创建新文件进行编辑
  • 文件存在则编辑已有文件

输入i进行编辑模式
按Esc退出编辑模式，回到命令模式
命令模式下输入:进入底线模式

一、命令模式（Normal Mode）

1. 光标移动

快捷键 作用
h j k l 左、下、上、右移动光标
w / b 跳到下一个/上一个单词开头
e 跳到当前单词末尾
0 / ^ 跳到行首（0是绝对行首，^是第一个非空字符）
$ 跳到行尾
gg 跳到文件第一行
G 跳到文件最后一行
5G 跳到第5行（数字可替换）
Ctrl + f / Ctrl + b 向下/向上翻页

2. 文本编辑

快捷键 作用
x 删除当前字符
dd 删除当前行
dw 删除当前单词
D 删除光标到行尾
yy 复制当前行
p / P 粘贴到光标后/前
u 撤销操作
Ctrl + r 重做（撤销的逆向操作）
>> / << 增加/减少缩进（当前行）

3. 搜索与替换

快捷键 作用
/pattern 向下搜索 pattern
?pattern 向上搜索 pattern
n / N 跳到下一个/上一个匹配项
:%s/old/new/g 全局替换 old 为 new

二、底线命令模式（Command-Line Mode）

进入方式：在命令模式下按 :
1. 文件操作

命令 作用
:w 保存文件
:wq / :x 保存并退出
:q! 强制退出（不保存）
:e filename 打开其他文件
:saveas path 另存为到指定路径

2. 窗口与分屏

命令 作用
:sp 水平分屏
:vsp 垂直分屏
:q 关闭当前窗口
:only 关闭其他窗口

3. 实用功能

命令 作用
:set nu 显示行号
:set nonu 隐藏行号
:noh 取消搜索高亮
:!command 执行外部命令（如 :!ls）

权限管理

切换用户

普通用户除了home目录大多数地方只有只读和执行权限，而root用户拥有最大的系统操作权限

语法：su [-] [用户名]

  • -符号可选，表示是否在切换用户后加载环境变量
  • 切换用户后可通过exit命令退回上一个用户，也可以通过ctrl+d退出

sudo命令

在命令前带上sudo，可为该命令临时赋予root权限

用户、用户组

Linux系统可以配置多个用户/多个用户组，Linux系统中关于权限管控有两个级别分别是用户级和用户组级

用户组管理

创建用户组：groupadd 用户组名
删除用户组：groupdel 用户组名

用户管理

创建用户：useradd [-g -d] 用户名

  • -g指定用户的组，不指定-g会自动创建同名的组，如果同名的组存在则必须用-g指定
  • -d指定用户Home路径，不指定默认在/home/用户名

删除用户：userdel [-r] 用户名

  • -r：删除用户的Home目录，不使用-r，删除用户时Home目录保留

查看用户所在组：id [用户名]

  • 不提供用户名则查看自身

修改用户所属组：usermod -aG 用户组 用户名

  • 将指定用户加入指定用户组

查看用户：getent passwd

  • 查看当前操作系统中有哪些用户

查看用户组：getent group

  • 查看系统全部系统组信息

权限

r代表读权限

  • 针对文件可以查看文件内容
  • 针对文件夹可以查看文件夹内容如ls命令

w代表写权限

  • 针对文件可以修改文件
  • 针对文件夹可以在文件夹内进行创建、删除、改名等操作

x代表执行权限

  • 针对文件表示可以将文件作为程序执行
  • 针对文件夹表示可以更改工作目录到此文件夹，及cd命令

chomd命令

用于修改文件、文件夹的权限信息

语法：chmod [-R] 权限 文件/文件夹

  • -R表示对文件夹内全部内容应用同样操作

chown命令

通过chown命令可以修改文件、文件夹所属的用户和用户组

注：普通用户无法修该所属为其他用户或组因此只适用于root用户执行
语法：chown [-R] [用户][:][用户组] 文件/文件夹

终端快捷键

软件安装

yum命令

rpm包管理器，用于自动化安装配置Linux软件并自动解决依赖问题，需要root权限

语法：yum [-y] [install | remove | search] 软件名称

  • -y：自动确认，无需手动确认安装或卸载流程
  • install： 安装
  • remove： 卸载
  • search：搜索

apt命令

deb包管理器，用于自动化安装配置Linux软件并自动解决依赖问题，需要root权限

语法：apt [-y] [install | remove | search] 软件名称

wget

通过sudo apt/yum install wget下载后进行使用

语法：wget [-O] https:...

  • -O：自定义保存路径和文件名，不写默认在当前目录下载，文件名与远程一致

ping

检查网络服务是否联通

语法：ping [-c num] ip或主机名

  • -c，检查的次数，不使用-c将无限次检查

curl

用于发起http网络请求，可用于下载文件、获取信息等

语法：curl [-O] url

  • -O用于下载文件

systemctl命令

用于控制软件的启动、停止、开机自启命令，被systemctl管理的软件一般称之为服务

语法：systemctl start|stop|status|enable|disable 服务名

  • start：启动服务
  • stop：暂停服务
  • status：查看服务状态
  • enable：设置开机自启
  • disable：禁用开机自启

系统内置服务如：

  • NetworkManager 主网络服务
  • network 副网络服务
  • firewalld 防火墙服务
  • sshd，ssh服务

软连接

在系统中创建软链接可以将文件、文件夹链接到其他位置，类似Windows的快捷方式

语法：ln -s 参数1 参数2

  • -s：创建软链接
  • 参数1：被链接的文件或文件夹
  • 参数2：要链接去的目的地

时间、时区

date命令

查看系统时间

语法：date [-d] [+格式化字符串]

  • -d：按照给定字符串显示日期，用于日期计算

支持格式：

  • 格式化字符串

进程管理

ps查看进程

语法：ps [-e -f]

  • -e：显示全部进程
  • -f:展示全部信息

kill关闭进程

语法：kill [-9] 进程ID

  • -9表示强制关闭
